# UniFloat Ad Script Generator

A PWA (Progressive Web App) that generates AI-powered WhatsApp video ad scripts for UniFloat — the campus fintech app for Ugandan university students.

## Features
- AI-generated ad scripts (powered by Claude)
- Multiple angles: Referral, Loans, Wallet, Games, Marketplace
- Multiple tones: Hype, Chill, Trustworthy
- Video lengths: 15, 30, 45 seconds
- One-tap copy to paste into HeyGen
- Installable as a phone app (PWA)

## Deploy to Vercel
1. Upload this folder to a GitHub repo
2. Go to vercel.com → New Project → Import your repo
3. Deploy — done!

## Install as Phone App
1. Open the deployed URL in Chrome on your phone
2. Tap 3-dot menu → "Add to Home Screen"
3. UniFloat appears on your home screen like a real app!
